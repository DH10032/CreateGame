# CreateGame


## 1차 목표

python으로 게임의 기본적인 틀 구현하기

1. tkinter

## 마무리 목표

wayland
linux api

이 두개의 api를 이용해.
게임의 그래픽을 띄운다

1. Wayland 아키텍처의 핵심 부분은 libwayland입니다. libwayland는 XML 프로토콜 정의를 C 언어 API로 변환하는 프로세스 간 통신 라이브러리입니다.
2. 사용자 공간에서 프로그램이 리눅스의 커널과 작용하기 위한 인터페이스 linux api를 이용한다.
3. 윈도우에서도 이와 같이 동작 시키기 위해 사용자의 운용체제를 파악 후, window인 경우, winapi를 이용한다.
